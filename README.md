# gilgamesj-new

## How to run ?

### admin gilgamesj

- npm i -g yarn
- yarn install
- yarn start-mac

### backend gilgamesj

- copy .env.local to .env (so we have 2 environment, .env.local and .env)
- npm i -g nodemon
- npm i -g yarn
- yarn install
- yarn start
